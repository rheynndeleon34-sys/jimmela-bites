## 📋 Overview

**Jimmela Food Products** is a full-stack food e-commerce and franchise management platform built for small food businesses specializing in frozen products like **Siomai** (Pork, Beef, Japanese, Shark's Fin) and **Longganisang Calumpit** (Regular, Garlic, Spicy, Sweet).

The platform features:
- **Public-facing website** for product advertising and franchise inquiries
- **Secure admin dashboard** with complete order, stock, and delivery tracking system
- **Real-time inventory management** with automatic stock updates
- **POS-like order processing** system for efficient business operations

### 🔒 Admin Dashboard (Management System)

#### Authentication
- **Secure Login** — Firebase Email/Password authentication
- **Protected Routes** — Unauthorized access redirects to login
- **Session Management** — Persistent login with Firebase Auth

#### Dashboard Overview
- **Key Metrics Cards** — Total orders, pending orders, total revenue, low stock alerts
- **Quick Stats** — Real-time business performance indicators

#### Products Management
- **View All Products** — Complete product catalog with images
- **Stock Management** — Update inventory levels in real-time

#### Orders Management
- **Order Tracking** — View all customer orders with details
- **Status Updates** — Update order status: Pending → Processing → Out for Delivery → Delivered → Cancelled
- **Order Details Modal** — View complete order information including items, customer details, and total
- **Order Statistics** — Track orders by status with count badges

#### Stock Management
- **Inventory Dashboard** — Real-time stock levels for all products
- **Low Stock Alerts** — Visual indicators for products running low
- **Stock Updates** — Edit stock quantities with instant database sync
- **Category Organization** — Products grouped by type (Siomai, Longganisa)

#### Delivery Tracking
- **Active Deliveries** — Track all orders in progress
- **Status Timeline** — Visual progress indicators for each delivery
- **Address Management** — Customer addresses with location details
- **Contact Information** — Customer phone numbers for delivery coordination
- **Real-time Updates** — Update delivery status instantly
